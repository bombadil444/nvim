
## Dependencies:
- ```pip install "python-lsp-server[all]"```
- ```sudo apt install ripgrep```

## Shortcuts:
### Project
- <space> pv = view files
- <space> pf = find file
- <space> ps = search text

### git
- <space> gs = git history
- ctrl + g = search git files

### Harpoon
- <space> a = add file
- ctrl + u = file 1
- ctrl + i = file 2
- ctrl + o = file 3
- ctrl + j = file 4

### Undo Tree
- <space> u = open file history
